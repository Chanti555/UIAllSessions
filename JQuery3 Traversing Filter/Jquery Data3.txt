7.Traversing:		
=============
	By default we can access dom elements in top-bottom approach only
	By using Traversing,we can access html dom elements in 3 approaches
	1.Traversing Up the DOM Tree
	2.Traversing Down the DOM Tree
	3.Traversing Sideways in The DOM Tree

1.Up the DOM Tree:
==================
parent()
parents()
parentsUntil()

2.Traversing Down the DOM Tree:
==============================
children()
find()

3.Traversing Sideways in The DOM Tree:
===================================
siblings()
next()
nextAll()
nextUntil()
prev()
prevAll()
prevUntil()
